Notas sobre o projeto

A ideia inicial era desenvolver um quiz que:

  • Tivesse uma identidade visual diferente do exemplo da aula.
  • Mostrasse o resultado das respostas ao usuário.
  • Boilerplate inicial fornececido (Bootstrap))

Considerações

Inicialmente, ponderei se faria o projeto utilizando o Bootstrap ou Tailwind. Optei pelo Bootstrap, por ser uma ferramenta que ainda é muito utilizada. E, provavelmente, vou encontrá-la sendo utilizada em projetos de clientes.

Passo a passo

Pesquisa: Primeiro, fui buscar inspirações para o design. Vi alguns sites de Quiz e peguei algumas referências.

CSS/Bootstrap: Comecei a ler a documentação do Boostrap e fiz o design do zero. Não utilizei o boilerplate inicial.

Html: Utilizei tags semânticas (article, aside, header, sections, etc.

Javascript: Depois, comecei a implementar o Javascript puro. (vanilla)

Funcionalidades adicionais

  • Efeito visual de sobreposição do mouse sobre as respostas. (ui/ux)
  • Expansão da área de clique para selecionar resposta. (ui/ux)
  • Validação do form. Necessário responder todas as perguntas antes de enviar o form. (array methods)
  • Informações adicionais sobre o tema da pergunta são inseridas após a conferência das respostas.(manipulação do DOM)
  • Rolagem da tela suave (ui/ux).
